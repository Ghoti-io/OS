error_code File::rename ( const std::string &  destinationPath)

Move or rename the file.

If the source file is a temp file, the destination will no longer be considered to be a temp file (meaning the file will not be automatically deleted when the object goes out of scope).

If the file is open, it will be automatically closed and must be re-opened explicitly.
